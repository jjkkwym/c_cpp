# videos

## you-get(视频下载工具)

### install

    pip install you-get

### info

[官网](https://you-get.org/)

[github](https://github.com/soimort/you-get)

对于高品质1080p视频you-get下载时可能会把视频和声音分开，
安装ffmpeg可自动合并它们(test in ubuntu20.04)

## ffmpeg(音视频处理)

### install

sudo apt install ffmpeg

### cmd

ffmped -i xxx.mp4 xxx.mp3
